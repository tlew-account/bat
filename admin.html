Roles & Access

Understand what each role can and cannot do.

👤 User Role

Standard athlete or coach account.

  • ✓ Access assigned apps only
  • ✓ View & edit their own records
  • ✓ Log sessions, create plans
  • ✗ Cannot see other users' data
  • ✗ Cannot access admin panel
  • ✗ Cannot promote or demote users

🔐 Admin Role ADMIN

Full platform access — assign carefully.

  • ✓ Access all apps
  • ✓ View ALL users' records
  • ✓ Promote or demote user roles
  • ✓ Manage app access per user
  • ✓ Send password reset emails
  • ✓ Access this admin panel

How to Promote Your First Admin

Run this in your Supabase SQL Editor after the user has registered:

UPDATE public.profiles SET role = 'admin'
WHERE email = 'your@email.com';

After that, all subsequent role changes can be made directly from this admin panel.

Password Administration

Send reset emails and manage passwords for any user.

📧 Send Password Reset Email

Triggers a Supabase magic reset link to the user's inbox. The link expires in 1 hour.

🔒 Force-Set Password

Directly sets a new password for a user. Run this SQL in your Supabase dashboard:

-- In Supabase SQL Editor:
SELECT auth.admin_update_user(
  'USER_UUID_HERE',
  'NewSecurePassword123!'
);
